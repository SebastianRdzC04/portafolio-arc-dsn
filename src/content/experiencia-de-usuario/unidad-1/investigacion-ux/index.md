## Introduccion

La experiencia de usuario (UX) se refiere a como una persona se siente al interactuar con un sistema, ya sea un sitio web, una aplicacion movil o cualquier producto digital. No se trata unicamente de la apariencia visual, sino de la facilidad de uso, la eficiencia y la satisfaccion general del usuario.

## Definicion de UX

Segun la norma **ISO 9241-210**, la experiencia de usuario abarca todas las respuestas y percepciones de una persona resultantes del uso anticipado o real de un producto, sistema o servicio.

Los elementos clave de UX incluyen:

- **Usabilidad**: Que tan facil es para el usuario lograr sus objetivos.
- **Accesibilidad**: Que el producto sea utilizable por la mayor cantidad de personas posible.
- **Deseabilidad**: Que la interfaz sea atractiva y genere una respuesta emocional positiva.
- **Utilidad**: Que el producto resuelva una necesidad real.

## Principios fundamentales

1. **Centrado en el usuario**: Toda decision de diseno debe partir de las necesidades reales de los usuarios.
2. **Iteracion continua**: El diseno UX es un proceso ciclico de investigacion, prototipado y evaluacion.
3. **Consistencia**: Los elementos de la interfaz deben comportarse de manera predecible.
4. **Retroalimentacion**: El sistema debe comunicar al usuario el resultado de sus acciones.
