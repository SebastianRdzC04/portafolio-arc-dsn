La evaluacion de accesibilidad es un proceso para verificar si los productos digitales o fisicos son utilizables para todas las personas, incluidad aquellas con discapacidades, para asegurar que cumplan con los estandares de accesibilidad. Se realizan mediante una combinacion de herramientas automaticas y pruebas manuales, que puedan incluir pruebas de navegacion con teclado, uso de lectores de pantalla y revision de aspectos visuales como el contraste de colores. 

## Tecnicas de Evaluacion de Accesibilidad de Interfaces
Las tecnicas para evaluar la accesibilidad de interfaces incluye auditorias automaticas y manuales, donde las herramientas automatizadas verifican problemas de codigo y los expertos realizan revisiones manuales para detectar problemas mas complejos. Otras tecnicas son la evaluacion euristica, las pruebas de usuarios reales la revision de expertos y los metodos de filtrado, como simular discapacidades. 

**Tecnicas Principales:**

* **Automatizacion de auditorias:** se utilizan herramientas automatizadas para verificar si el codigo cumple con las pautas de accesibiidad, como el contraste de color y la semantica del marcado. Son rapidas y utiles para detectar errores de codigo pero no son suficientes por si solas.
* **Autidorias Manuales:** Realizadas por expertos de accesibilidad, estas pruebas son mas detalladas y exaustivas ya que se centran en el problema de usabilidad y diseño que las herramientas automaticas no pueden detectar.
* **Pruebas con Usuarios:** Involucran a personas con discapacidades **reales** que utilizan las interfaces para evaluar a experiencia de usuario en situaciones practicas. 
* **Evaluacion Euristica:** se basa en principios euristicos de accesibilidad para inspeccionar la interfaz y detectar posibles problemas.
* **Revision por Expertos:** Similar a la auditoria manual implica la opinion de expertos en accesibilidad que pueden identificar barreras y ofrecer recomendaciones
* **Filtrado** simula dificutades para que los evaluadores enfrenten los desafios que enfrentan los usuarios. Ejemplos incluyen usar guantes gruesos para limitar la destreza o un lector de pantalla sin usar el monitor
* **Tutorial Cognitivo:** Es un enfoque que examina la interfaz para evaluar la carga cognitiva y la caridad de a informacion, especialmente util para usuarios con discapacidades cognitivas
